# Marine Team Website

เว็บแนะนำสมาชิกทีม Marine -- สร้างโดย Team Marine เอง

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- TypeScript
- Bun

## Development

```bash
bun install
bun dev
```

## Build

```bash
bun run build
```

Output will be in the `out/` directory (static export).
